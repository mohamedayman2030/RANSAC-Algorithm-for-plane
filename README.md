# RANSAC-Algorithm-for-plane
implement RANSAC algorithm to find the outliers in plane using c++
## the Algorithm
RANSAC stands for Random Sample Consensus, and is a method for detecting outliers in data
### build steps
cd build
cmake ..
make
./quizRansac
